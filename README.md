# Travel Analytics Dashboard

## Overview
This project is an end-to-end travel data analysis project built using SQL, Microsoft Excel, and Power BI. It demonstrates how travel booking data can be created, analyzed, and visualized to uncover trends, customer segments, and business insights.

## Tools Used
- SQL
- Microsoft Excel
- Power BI Desktop

## Project Files
- `travel_dataset_setup.sql` – SQL script used to create and structure the dataset
- `travel_analysis_queries.sql` – SQL queries used to analyze travel data
- `Luxury_Travel_Analysis.xlsx` – Excel file containing the travel dataset and pivot chart analysis
- `Luxury_Travel_Analytics_Dashboard.pbix` – Interactive Power BI dashboard
- `Luxury_Travel_Analytics_Dashboard.pdf` – Exported dashboard report

## Key Analysis Areas
- Top destinations by number of bookings
- Average travel package cost by travel type
- Monthly booking trend analysis
- Booking distribution by age group
- Special requests analysis

## Dashboard Features
- Interactive slicers for Destination, Travel Type, and Age Group
- Bar chart showing top destinations
- Column chart showing average package cost by travel type
- Line chart showing monthly booking trends
- Pie chart showing booking distribution by age group
- Bar chart showing customer special requests

## Skills Demonstrated
- Data extraction and manipulation using SQL
- Data analysis using Excel
- Dashboard creation and data visualization in Power BI
- Trend analysis and reporting
- Business-focused storytelling with data

## Purpose
This project was created to showcase practical business intelligence and data analysis skills through a travel-focused use case. It reflects an end-to-end workflow from dataset creation and querying to reporting and dashboard design.
