# 📊 Travel Data Analysis & Visualization (Excel + SQL + Power BI)

---

## 📌 Project Overview
This project focuses on analyzing travel booking data to identify patterns, trends, and insights that can support data-driven decision-making. The analysis includes data cleaning, transformation, exploratory data analysis (EDA), and interactive dashboard creation.

---

## 🎯 Problem Statement
Travel companies generate large volumes of booking data. The goal of this project is to:
- Identify customer behavior patterns
- Analyze seasonal booking trends
- Discover high-performing destinations
- Provide insights for business strategy and forecasting

---

## 🛠️ Tools & Technologies
- **Excel** → Data cleaning & preprocessing  
- **SQL** → Data extraction & analysis  
- **Power BI** → Data visualization & dashboard  

---

## 🧹 Data Cleaning & Preprocessing
- Removed null/missing values to ensure data quality  
- Eliminated duplicate records  
- Standardized date formats and categorical values  
- Structured raw data for efficient querying and analysis  

---

## 📈 Exploratory Data Analysis (SQL)
Performed data analysis using SQL to uncover patterns and trends:
- Used **JOINs** to combine multiple datasets  
- Applied **aggregations** (SUM, AVG, COUNT)  
- Identified:
  - Peak booking periods  
  - Most popular destinations  
  - Revenue trends over time  
  - Customer booking behavior  

---

## 📊 Power BI Dashboard
The dashboard provides an interactive view of key metrics:

### 🔹 Features:
- KPI Cards (Total Revenue, Total Bookings, etc.)
- Filters/Slicers (Region, Time, Category)
- Trend analysis (monthly & seasonal)
- Comparative insights across locations
